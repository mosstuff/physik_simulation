## Annahmen / Vereinfachungen
### Annahmen
- Drei masselose Stäbe als Ideale Pendel
- Punktmasse an Pendelspitze
- Pendel mit Federn an Punktmassen verbunden
- Euler-Methode mit dt=0.005s

### Vereinfachungen
- Keine Reibung
- Federn haben keine Massen
- Drehpunkte sind fixiert (keine Bewegung der Aufhängung)
- 2D-Bewegung

### Parameter
- Parameter 1 (veränderbar): Federkonstante k
- Parameter 2 (veränderbar): Startwinkel des mittleren Pendels

### Was kann das Modell gut?
- Zeigt was chaos sein kann
- Relativ simpel zu verstehen

### Wo scheitert es?
- Euler-Methode ist Fehleranfällig (es kann neue Energie entstehen)
- Federn haben auch eine zu simulierende Masse und Gewicht
- Ideales System -> nicht in der Realität möglich
